一个专业的股票分析系统，支持基本面分析、技术分析、行业特定分析和实时市场数据，提供命令行和Web两种使用方式。

## ✨ 核心特色

### 📊 全面分析能力
- **基本面分析**：5大行业专用分析模型（软件、金融、能源、医疗、工业）
- **技术面分析**：30+技术指标（RSI, MACD, 布林带等）
- **宏观数据**：利率、通胀、就业等经济指标
- **智能行业检测**：自动识别股票所属行业并应用对应分析规则

### 🌐 双模式使用体验
- **📋 命令行模式**：生成专业TXT分析报告，适合存档和分享
- **🎨 Web界面模式**：交互式仪表板，实时图表和可视化分析

### ⚡ 智能功能
- **💾 智能缓存系统**：6小时缓存有效期，大幅提升重复分析速度
- **🏭 行业适配**：不同行业使用不同的估值和风险评估标准
- **📈 实时数据**：从Yahoo Finance获取最新股价、新闻和财务数据
- **🎯 交易信号**：量化模型生成投资建议和信心指数
